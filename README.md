# Veloce Web Studio 🚀

A premium, highly responsive static agency website designed to showcase web development services and seamlessly capture client project briefs. Built with performance and user experience in mind, this landing page features a modern UI and direct client-to-agency communication routing.

## 🔗 Live Demo
**[Experience the live site here](https://nithilan-su57.github.io/SellWebsite/)**

## ✨ Features
* **Modern UI/UX:** Clean, dark-themed aesthetic utilizing Tailwind CSS utility classes for a fully responsive design across all devices.
* **Smart Lead Capture:** A custom-built project scoping form that captures client details and package preferences.
* **Dual-Routing System:**
  * **WhatsApp Integration:** Dynamically constructs and redirects to a pre-filled WhatsApp message for instant client communication.
  * **Email Routing:** Integrates with Web3Forms API to silently deliver client briefs directly to the agency's inbox without requiring a backend server.
* **High Performance:** Lightweight static assets (HTML/CSS/JS) ensuring fast load times and zero-dependency deployments.

## 🛠️ Tech Stack
* **Markup:** HTML5
* **Styling:** CSS3 / Tailwind CSS (via CDN/compiled)
* **Scripting:** Vanilla JavaScript (ES6+)
* **Hosting:** GitHub Pages
* **APIs:** Web3Forms (Email Routing), WhatsApp Web API
